# Bot Functional Auditor

This skill provides a comprehensive diagnostic of the bot's internal state and logic execution.

## Usage

Run the master audit script:

```bash
./.agent/skills/bot_functional_auditor/scripts/full_audit.sh
```

Run maintenance tasks:

```bash
# Production Pipeline (Data + Models + Weights)
./.agent/skills/bot_functional_auditor/scripts/run_maintenance_prod.sh

# System Maintenance (Service Restart + Backfill + GA)
./.agent/skills/bot_functional_auditor/scripts/run_maintenance_system.sh
```

## Audit Checks
- **Process Check:** Verifies if the C++ binary is active.
- **Signals Audit:** Checks the frequency and quality of signals generated by the Ensemble (Markov/RL).
- **Risk Audit:** Analyzes current correlation levels and time-exits.
- **Execution Audit:** Compares DB trades vs Bybit API status to ensure no "ghosts" remain.
- **Maintenance Audit:** Verifies the last execution of the maintenance pipeline (retraining/GA).
- **Database Health:** Verifies the `trading_data.db` integrity.

Use this when you suspect the bot has "frozen" or if you want to verify that all the new fixes (Hedge Mode, Time-Exit) are actually working in sync.
